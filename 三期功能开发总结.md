## 核心功能说明

### 1. 绘本阅读功能
- **自动播放模式**：自动朗读文字并翻页
- **手动翻页模式**：家长陪伴模式，手动控制
- **锁定模式**：防止宝宝误触翻页
- **阅读进度**：记录当前页码、播放模式、锁定状态
- **TTS 朗读**：调用阿里云 TTS 实时合成语音

### 2. 管理端功能
- **绘本管理**：新增、编辑、删除、上下架
- **页面管理**：逐页上传图片和文字
- **页面排序**：调整页面顺序

### 3. 数据存储
- **绘本数据**：存储在 PostgreSQL
- **阅读进度**：存储在 Redis（180天过期）
- **TTS 音频**：临时生成，前端缓存

## 注意事项

1. **阿里云 TTS 配置**
   - 当前是简化实现，返回模拟数据
   - 实际使用需要配置阿里云 AccessKey 和 AppKey
   - 需要添加阿里云 TTS SDK 依赖

2. **图片存储**
   - 建议使用对象存储 + CDN
   - 确保图片 URL 可访问

3. **音频缓存**
   - 前端使用 Map 缓存已生成的音频
   - 页面卸载时清理缓存

4. **兼容性**
   - 测试移动端浏览器兼容性
   - 测试 TTS 音频播放
